# Digital Soseki 2

タブレット＋ペンで青空文庫テキストを **TEI XML** にタグ付けするための、単一ファイル Web アプリです。
本体は `index.html` 一つに HTML/CSS/JS をすべて内包しています（外部依存は xlsx 読み込み用の SheetJS CDN のみ）。

Python で処理する従来システム（v1）の後継にあたるブラウザ版です。Wordの背景色機能を使って、手作業でタグの範囲と種類をマークしていたワークフローを、ブラウザ上で選択 → タグ確定 → TEI 書き出しの流れに置き換えます。

## 使い方

1. **読み込み** … 青空文庫のプレーンテキスト（Shift-JIS / UTF-8 自動判定）または既存の TEI XML を読み込みます。青空文庫のテキストを読み込むときは、ヘッダ・`［＃…］`注記を除去し、外字 `※［＃…第3水準1-88-22］` は JIS-Unicode テーブルで変換します。
2. **タグ付け** … 本文を**ドラッグで選択**してから、上のパレットでタグを選ぶと確定します。本文の色付き部分を**クリック／タップ**するとタグを編集できます。主題（seg）は複数分類を選べます（`ana="#a #b"`）。
3. タグ一覧は、タグの種類やキーワードで絞り込むことができます。
4. 作業中に記録しておきたい部分にブックマークをつけることができます。ブックマークはXMLとして書き出す際には無視されます。
5.  **書き出し** … TEI XML を出力します。編集中のファイルを保存しておけば、再読み込みをして編集を続けられます。

タグの定義は **色／タグ／ラベル／属性／属性値** の5列の Excel/CSV で差し替え可能です。一度読み込めばブラウザのローカル・ストレージに自動保存され、本文・外字テーブルとあわせて次回も保持されます。主題（seg）タグは手動で追加することもできます。

## 公開（GitHub Pages）

https 配信にすると、`file://` 直開きで起きる localStorage 無効や、オフライン時の xlsx CDN 失敗を回避できます。タブレットからは公開 URL を開くだけで使えます。

> Settings → Pages → Branch を `main` / `(root)` に設定。公開後の URL をここに記載しておくと便利です。
> 例：`https://<ユーザ名>.github.io/<リポジトリ名>/`
