# Digital Soseki 2

タブレット＋ペンで青空文庫テキストを **TEI XML** にタグ付けするための、単一ファイル Web アプリ。
本体は `index.html` 一つに HTML/CSS/JS をすべて内包しています（外部依存は xlsx 読み込み用の SheetJS CDN のみ）。

Python で処理する従来システム（v1）の後継にあたるブラウザ版です。背景色で手作業マークしていたワークフローを、選択 → タグ確定 → TEI 書き出しの流れに置き換えます。

## 使い方

1. **読み込み** … 青空文庫のプレーンテキスト（Shift-JIS / UTF-8 自動判定）または既存の TEI XML を読み込む。読み込み時にヘッダ・`［＃…］`注記を除去し、外字 `※［＃…第3水準1-88-22］` は JIS-Unicode テーブルで変換します。
2. **タグ付け** … 本文を**ドラッグで選択** → 上のパレットでタグを選ぶと確定。色付き部分を**タップ**すると編集できます。主題（seg）は複数分類を選べます（`ana="#a #b"`）。
3. **書き出し** … TEI XML を出力。保存しておけば再読み込みして編集を続けられます。

タグ定義は **色／タグ／ラベル／属性／属性値** の5列の Excel/CSV で差し替え可能。一度読み込めば自動保存され、本文・外字テーブルとあわせて次回も保持されます。

## 公開（GitHub Pages）

https 配信にすると、`file://` 直開きで起きる localStorage 無効や、オフライン時の xlsx CDN 失敗を回避できます。タブレットからは公開 URL を開くだけで使えます。

> Settings → Pages → Branch を `main` / `(root)` に設定。公開後の URL をここに記載しておくと便利です。
> 例：`https://<ユーザ名>.github.io/<リポジトリ名>/`
